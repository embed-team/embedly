The **Embed Links** context menu command lets you trigger an embed on any message that was already posted — useful if Embedly missed a link, or the message predates Embedly being in the server.

## Usage

Right-click any message (or long-press on mobile), then go to **Apps → Embed Links**.

Embedly will find the first supported link in that message and post an embed.

<Callout type="info">
  Context menu embeds don't support flags like `media_only` or `source_only`. Use the [slash command](/docs/embedly/slash-command) if you need those.
</Callout>
