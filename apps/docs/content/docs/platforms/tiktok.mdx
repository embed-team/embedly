All TikTok URL formats are supported, including short links. Embedly follows redirects to resolve the canonical video URL before fetching.

## Supported URLs

```
https://www.tiktok.com/@<user>/video/<id>
https://vm.tiktok.com/<short>
https://m.tiktok.com/...
```

## Supported features

### Text

The video caption.

### Media

The video.

### Stats

Comments, shares, likes, and views.
